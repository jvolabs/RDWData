## Applied redesign decisions
- Brand-only header for cleaner focus and less navigation noise.
- Hero restructured into a split layout:
  - left: headline + search action
  - right: compact value panel
- Visual hierarchy tightened using:
  - stronger title scale
  - secondary muted text
  - clear status badges
- Section cards upgraded with softer elevation, larger radius, and controlled accent gradients.
- Result page shifted to an editorial data layout with a dedicated summary panel.

## UX outcomes targeted
- Faster first action (plate input is dominant)
- Better scanability of lookup state and key metrics
- Reduced clutter for MVP while preserving expansion room
